###########################
Ndigo6G |hyphen| User Guide
###########################

.. note:: 

    This User Guide is under active developement and is subject to change.

The `Ndig6G-12 <https://www.cronologic.de/product/ndigo6g-12>`_
by `cronologic GmbH & Co. KG <https://www.cronologic.de>`_
is a hybrid ADC/TDC solution for the acquisition of short pulses.

This User Guide documents the Driver Programming API provided by the
Ndigo6G-12 driver.
