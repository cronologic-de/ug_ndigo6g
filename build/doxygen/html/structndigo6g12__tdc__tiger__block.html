Detailed Description

Contains settings of TiGer block.

The configuration is similiar to ndigo6g12_gating_block.

Member Data Documentation

◆ mode

int mode

Enables the desired mode of operation for the TiGeR.

It is of one of the values NDIGO6G12_TIGER_* .

Default is NDIGO6G12_TIGER_OFF.

◆ negate

crono_bool_t negate

Set pulse polarity.

The TiGeR creates a high pulse from start to stop unless negated.

Default is true.

◆ retrigger

crono_bool_t retrigger

Enable retriggering.

If enabled:

  • If the timer surpassed the time corresponding to start, it is reset to start.
  • If the timer did not surpass the time corresponding to start, it is reset to zero.

Otherwise signals at the input sources are ignored until stop is reached.

Defaults to false.

◆ sources

int sources

Bit mask with a bit set for all trigger sources that can trigger this channel.

Default NDIGO6G12_TRIGGER_SOURCE_<channel>0 (NDIGO6G12_TRIGGER_SOURCE_A0 for ADC channel A, NDIGO6G12_TRIGGER_SOURCE_B0 for ADC channel B, etc).

◆ start

int start

The time from the first input signal seen in the idle state until the TiGer outputs a signal.

In multiples of 5 ns. 0 ≤ start < 216, while start ≤ stop.

Default is 0.

◆ stop

int stop

The number of samples from leaving the idle state until the TiGer output is reset.

In multiples of 5 ns. 0 ≤ stop < 216, while stop ≥ start.

Default is 1.
